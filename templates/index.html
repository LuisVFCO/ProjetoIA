Bem-vindo ao Sistema de Detecção de Ocupação

Este aplicativo foi desenvolvido para identificar, de forma rápida, se um ambiente está ocupado ou vazio, com base em dados coletados anteriormente.

Utilizando técnicas de inteligência artificial, o sistema analisa variáveis como temperatura, umidade, luminosidade, nível de CO₂ e razão de umidade para gerar uma predição confiável.

Você pode inserir os dados manualmente e visualizar os resultados em tempo real, além de explorar gráficos interativos com estatísticas das previsões já realizadas.
